# ACC102--Mini-Assignment
IBM Competitors Analysis(2015-2024)
### Track 2: GitHub Data Analysis Project

### Project Overview
This project completes quantitative financial analysis on IBM's annual net income over the fiscal year period from 2015 to 2024.
Data is retrieved from WRDS Compustat database, with Python data processing, visualization and statistical analysis to explore the company's profitability performance and earnings trend.

### Target Audience
Investment analysts, finance students and individual retail investors.

### Tools & Libraries Used
- Python
- WRDS API
- Pandas (data cleaning & processing)
- Matplotlib (data visualization)

### Analysis Contents
1.  WRDS database data query and data extraction
2.  Data pivoting, unit conversion and data cleaning
3.  Annual profit bar chart visualization
4.  10-year profit trend line chart analysis
5.  Descriptive statistical summary and financial insight conclusion

### Data Source
WRDS Compustat fundamental database (funda table)
Access Date: April 2026

### AI Disclosure
AI tools were used for auxiliary support including Python code debugging, error fixing and English report polishing. All data analysis logic, result interpretation and final conclusions are independently completed and verified by the author.
