# ACC102--Mini-Assignment
IBM Competitors Analysis(2015-2024)
ACC102 Track 2 GitHub Data Analysis Project

## 1. Problem & User
This project analyzes IBM’s annual net income performance from 2015 to 2024.
The intended users are finance students, investment analysts, and general investors.

## 2. Data
- Source: WRDS Compustat Database
- Access date: April 2026
- Key field: Net Income (annual)
- Period: 2015–2024

## 3. Methods
- Python data retrieval via WRDS library
- Data cleaning and unit conversion with Pandas
- Visualization with Matplotlib (bar chart & line chart)
- Descriptive statistical analysis

## 4. Key Findings
- IBM achieved the highest net income in 2015.
- The lowest net income occurred in 2022 due to business restructuring and macroeconomic pressure.
- Profit gradually recovered in 2023 and 2024.
- All yearly net income values remained positive.

## 5. How to Run
1. Open the Jupyter notebook file (.ipynb)
2. Run all code cells sequentially
3. View tables, charts, and summary outputs directly in the notebook
